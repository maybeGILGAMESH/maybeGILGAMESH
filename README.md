## about me

I build practical AI systems for document understanding, retrieval-augmented generation, computer vision and security automation.

My main focus is turning research ideas into working pipelines: local LLM/VLM inference, multi-page document processing, evaluation, APIs and production-ready prototypes.

## focus areas

- retrieval-augmented generation for real documents
- multimodal models for multi-page document analysis
- local LLM and VLM systems without external cloud dependency
- document AI, OCR-free extraction and structured outputs
- LLM evaluation, guardrails and red-team testing
- computer vision, detection and remote sensing
- security automation and smart-contract analysis
